# Contributing to Baar-Core

First off, thank you for considering contributing to Baar-Core! It's people like you that make the agentic future accessible and safe for everyone.

## 🌈 How Can I Contribute?

### Reporting Bugs
- Use the [Bug Report Template](https://github.com/orvi2014/Baar-Core/issues/new?template=bug_report.md).
- Provide a clear, descriptive title.
- Include steps to reproduce the issue.

### Suggesting Enhancements
- Use the [Feature Request Template](https://github.com/orvi2014/Baar-Core/issues/new?template=feature_request.md).
- Explain why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (`pytest`).
5. Submit a pull request!

## 🛡️ Security Vulnerabilities
If you discover a security vulnerability, please send an e-mail to **das.orvi@yahoo.com**. We will address it immediately.

## 🛠️ Development Setup
1. Clone the repository.
2. Install dev dependencies: `pip install -e ".[dev]"`
3. Run tests: `pytest`

## 📜 Code of Conduct
By participating in this project, you agree to abide by our code of conduct. We are committed to providing a welcoming and inspiring community for all.
